# Spensa - Money Tracker & Budget Manager

Spensa is a **money tracker app** designed to help users **manage their wallets, track expenses, and optimize budgets** efficiently. With **AI-powered receipt scanning**, users can easily upload receipts and extract data without manual entry. Spensa aims to provide a **seamless, free-to-use** experience without subscription-based services.

## Features

- **Expense Tracking:** Monitor your daily, weekly, and monthly spending.
- **Wallet Management:** Add multiple wallets to track balances and transactions.
- **Budget Planning:** Set financial goals and limits to manage spending effectively.
- **AI Receipt Scanning:** Upload receipts, and AI will extract transaction details automatically.
- **Free to Use:** No hidden fees or subscriptions.
